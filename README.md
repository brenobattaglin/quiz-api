# quiz-api

## Description

A quiz API made with NestJS that uses MongoDB.

## Installation

```bash
$ npm install
```

## Running the app

Update the _src/app.module.ts_ with the MongoDB connection string, then run one of the following commands

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
